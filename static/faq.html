Rewind Frequently Asked Questions

  • Is Rewind GDPR compliant?

    • Yes. Rewind does not store any identifiable information so it is completely GDPR compliant.

  • How many customer replays can I store per month?

    • Unlimited! Rewind is highly engineered to bring you a powerful yet cheap analytical tool.

  • Will Rewind leave bloat in my website?

    • The only thing that Rewind installs is a script tag. Script tags are automatically uninstalled with the app so no trace is left behind.

  • Does Rewind record passwords from customers?

    • No. Rewind censors all inputs of type "password".
